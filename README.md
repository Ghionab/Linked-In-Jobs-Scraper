# LinkedIn Job Scraper

Simple Python app to scrape LinkedIn jobs with a clean GUI.

## Quick Start

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

2. Run the app:
   ```bash
   python main.py
   ```

3. Enter job title and location, click Search

## Features

- Clean PyQt5 interface
- LinkedIn job scraping
- Job status tracking
- Anti-detection measures

## Requirements

- Python 3.7+
- Chrome browser
- Internet connection
